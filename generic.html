Muscle-Fit Tracker (Website)

Muscle-Fit Tracker is a concept fitness tracker app that allows a user to log his/her workouts by muscle groups worked. This allows the user to distribute workouts in a balanced manner. This website is essentially an advertisement for the app's release. The website also offers calulators for various fitness-related metrics such as BMI (Body-mass Index). Furthermore, a sign-up form is available for those interested in learning more about the app's release in the future.

The website contents were developed using HTML5, CSS3 and JavaScript. Bootstrap was the framework chosen for a light and responsive front-end structure. The form requests utilize PHP.

This website was created for the final project in the following course: CS 329E (Elements of Web Design). This course was offered in Fall 2018 and taught by Dr. Mitra at the University of Texas at Austin. This was a group project with the following team members: Shakkhar Biswas, Shane Mathew, Rohan Chaudhry.
